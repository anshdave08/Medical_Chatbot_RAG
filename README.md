Medical RAG Chatbot
Overview
AI-powered medical chatbot using Retrieval-Augmented Generation (RAG).

Features
Disease information (symptoms, causes, treatment)
Uses vector database for retrieval
Reduces hallucination
Tech Stack
Python
FAISS / ChromaDB
LLM (Groq/OpenAI)
LangChain
Setup
Clone repo

Install requirements: pip install -r requirements.txt

Add API key in .env

Run: python app.py
