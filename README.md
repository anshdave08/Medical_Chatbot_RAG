# Medical RAG Chatbot

## Overview
AI-powered medical chatbot using Retrieval-Augmented Generation (RAG).

## Features
- Disease information (symptoms, causes, treatment)
- Uses vector database for retrieval
- Reduces hallucination

## Tech Stack
- Python
- FAISS / ChromaDB
- LLM (Groq/OpenAI)
- LangChain

## Setup

1. Clone repo
2. Install requirements:
   pip install -r requirements.txt

3. Add API key in .env
4. Run:
   python app.py
